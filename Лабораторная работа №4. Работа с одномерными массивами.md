1. **Постановка задания:**

   12 вариант.

   1. Сформировать одномерный массив целых чисел, используя датчик случайных чисел.
   1. Распечатать полученный массив.
   1. Поменять местами минимальный и максимальный элементы массива.
   1. Удалить из массива все элементы превышающие его среднее значение более, чем на 10%.
   1. Распечатать полученный массив.

1. **Анализ решения:**
1) Для наполнения одномерного массива случайными числами в диапазоне от -50 до 49 используем функцию rand(). 
1) После объявления и наполнения массива случайными числами выводим массив в консоль для того, чтобы пользователь мог увидеть исходный массив и сравнить его с итоговым массивом после проделанных операций над ним.
1) Для перестановки местами минимального и максимального элемента массива используем цикл, который проходит по всем элементам массива. В цикле используем два условных оператора, позволяющих определить и запомнить значение и индекс максимального и минимального элемента массива.

   После завершения цикла происходит обмен значений максимального и минимального элемента массива через переменную temp.

1) Для того, чтобы удалить из массива все элементы, превышающие его среднее значение более, чем на 10%, при помощи цикла считаем сумму всех элементов и записываем в переменную sum. Далее вычисляем среднее арифметическое массива при помощи формулы sum / n, где сумма элементов делится на количество элементов массива и записываем значение в avg. Затем задаем порог limit = avg \* 1.1. Все элементы, которые больше этого порога, удаляются. В условиях статического массива физического удаления нет, но применяется псевдоудаление - подходящие элементы перезаписываются в начало массива в том же порядке, а их количество учитывается отдельной переменной.
1) Выводятся только те элементы, которые остались после фильтрации.
